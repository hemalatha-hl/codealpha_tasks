# Hangman Game

## Description
This is a simple text-based Hangman Game developed using Python. The player guesses a hidden word one letter at a time within a limited number of attempts.

## Features
- Random word selection from a predefined list
- Maximum of 6 incorrect guesses
- Displays guessed letters and remaining attempts
- Input validation for single-letter entries
- Prevents repeated guesses

## Technologies Used
- Python

## How to Run
1. Install Python
2. Open terminal in the project folder
3. Run the command:
   python hangman.py

## Output
The game runs in the console and displays the word progress along with remaining attempts.
