# Stock Portfolio Tracker

## Description
This project is a simple stock portfolio tracker developed using Python. It allows users to input stock names and quantities, then calculates the total investment value based on predefined stock prices.

## Features
- User input for stock names and quantities
- Predefined stock price dictionary
- Calculates total investment value
- Displays portfolio summary
- Option to save results to a text file

## Technologies Used
- Python

## How to Run
1. Install Python
2. Open terminal in the project folder
3. Run:
   python stack_portfolio_tracker.py

## Output
Displays stock details and total investment in the console. Optionally saves results to a file.
